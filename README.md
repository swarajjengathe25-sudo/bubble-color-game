# Bubble Color Game 🎨

A simple interactive game built using HTML, CSS, and JavaScript.

The player has to click the bubble that matches the target color shown on the screen.

## Features

* Random colored bubbles
* Target color matching
* Interactive gameplay
* Simple and clean user interface

## Technologies Used

* HTML
* CSS
* JavaScript
